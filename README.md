# FrequencyCalculator

备注：
1、使用本代码编译出exe程序，需要将logo、theme、translations三个文件夹以及数据库文件customdb.db放在同一级目录中；
2、程序中使用的频段与数据库文件customdb.db文件同步，数据库中新增一个频段信息，重启本程序后会自动添加该频段；
